NJU-EL：穿进《红楼梦》后，我成了甄家大小姐！？

## 项目简介

这是一款基于团结引擎开发的 WebGL 互动式文化学习游戏。作品以《红楼梦》、南京地方文化、江宁织造、云锦等元素为灵感来源，通过视觉小说式剧情、场景跳转、多存档选择、智能体入口和地图入口等方式，尝试构建一个轻量、可交互、可学习的传统文化体验空间。

当前版本为序章展示版。该版本主要完成了主菜单、序章剧情、多存档选择界面、智能体对话入口、地图入口以及 WebGL 在线部署。后续版本将继续完善章节探索、背包系统、物品卡片、文化谜题和智能体知识问答等内容。

项目主题为：

> **Easy Learning, Easy Life**

我们希望通过网页化、游戏化和轻量化的方式降低传统文学与地方文化学习门槛，让玩家无需安装 Unity 或额外客户端，只需要打开网页，就可以进入一个围绕《红楼梦》与南京文化构建的互动世界。

## 操作说明

### 主菜单

进入游戏后，玩家会看到主菜单界面。主菜单包含：
开始游戏
读取存档
对话
地图

### 开始游戏

点击“开始游戏”后，默认从存档一开始新游戏，并进入序章剧情。

### 读取存档

点击“读取存档”后，进入存档选择界面。

当前支持三个存档槽

点击对应存档的“继续游戏”按钮后，系统会读取或创建该槽位的进度。

### 对话

点击“对话”后进入智能体对话入口。玩家可与红楼经典人物对话，后续可问关于《红楼梦》、南京文化、云锦、江宁织造等内容的问题。

### 地图

点击“地图”后进入地图展示入口。玩家可通过点击文化地标获取相应地点文化知识。


在 WebGL 网页版中，存档数据保存在浏览器本地。因此需要注意：
1. 同一浏览器中可以正常读取存档
2. 更换浏览器后可能没有原存档
3. 清理浏览器缓存后存档可能消失
4. 无痕模式下存档可能不稳定

该功能用于比赛展示和轻量体验，后续版本可以继续扩展为更完整的存档系统。


## 项目亮点

### 1. 传统文化学习游戏化

本项目尝试将《红楼梦》和南京地方文化转化为可体验、可互动的游戏内容，使传统文学学习不再停留在静态阅读层面。

### 2. WebGL 降低体验门槛

通过 WebGL + GitHub Pages 部署，玩家和评委无需安装 Unity / 团结编辑器，只需打开网页即可体验作品。

### 3. 多存档槽设计

当前版本支持存档一、存档二、存档三，便于玩家保存不同进度，也为后续多分支剧情扩展提供基础。

### 4. 智能体入口预留

项目中设置了智能体对话入口，后续可进一步接入文化知识问答、剧情提示和学习辅助功能。

### 5. 地图入口预留

地图入口为二周目探索、地点回顾和知识复盘提供了基础框架。

我们为什么想到做这个游戏————
传统文学与历史文化学习常常存在以下问题：
内容抽象
背景复杂
资料分散
人物关系较多
学习过程缺少即时反馈

本项目希望通过游戏化方式降低这些门槛：
用剧情代替生硬讲解
用点击代替被动阅读
用存档系统支持自由体验
用智能体入口提供即时解释方向
用地图入口支持后续知识复盘
用 WebGL 网页部署降低使用成本

玩家不需要安装复杂软件，也不需要先阅读大量资料，只需要打开网页，就可以进入围绕《红楼梦》与南京文化构建的互动世界。

这正是本项目对 **Easy Learning, Easy Life** 的理解：

学习可以更轻松。
文化可以更亲近。
游戏可以成为知识进入生活的入口。
